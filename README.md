# Zero Launcher

A lightweight, customized Minecraft launcher.

---

## English Version

### About the Project
I created this launcher because after spending a long time searching for the perfect Minecraft launcher, I couldn't find one that completely suited my needs. To solve this, I built Zero Launcher from scratch with the assistance of AI.

### Requirements
To run this launcher, you need Java with JavaFX support:
* Azul Zulu JDK FX: https://www.azul.com/downloads/?package=jdk-fx#downloads-table-zulu

### Troubleshooting
If the launcher stops working after the "Checking for updates" screen:

* Windows:
  1. Press Win + R, type %appdata% and hit Enter.
  2. Open the .zerolauncher folder.
  3. Delete zerolauncher_1_x_x.jar.

* Linux:
  1. Go to /home/.zerolauncher.
  2. Delete zerolauncher_1_x_x.jar.

## النسخة العربية

### عن المشروع
قمت بتطوير هذا اللانشر لأنني قضيت فترة طويلة أبحث عن لانشر ماينكرافت يناسبني ولكنني لم أجد ما يرضيني. بناءً على ذلك، قمت ببرمجة Zero Launcher بمساعدة الذكاء الاصطناعي (AI).

### المتطلبات
لتشغيل اللانشر ستحتاج إلى جافا يدعم JavaFX:
* Azul Zulu JDK FX: https://www.azul.com/downloads/?package=jdk-fx#downloads-table-zulu

### حل المشاكل
إذا توقف اللانشر عن العمل بعد ظهور شاشة Checking for updates:

* على نظام Windows:
  1. اضغط Win + R واكتب %appdata% ثم اضغط Enter.
  2. افتح مجلد .zerolauncher.
  3. احذف ملف zerolauncher_1_x_x.jar.

* على نظام Linux:
  1. انتقل إلى المسار /home/.zerolauncher.
  2. احذف ملف zerolauncher_1_x_x.jar.
