# Zero Launcher

A modern Minecraft launcher built entirely with Java (Swing/AWT), designed to be fast, lightweight, and highly customizable. Zero Launcher focuses on making instance and mod management simple while providing an interface that feels clean, responsive, and enjoyable to use.

## Features

### Multiple Minecraft Instances

* Create, edit, duplicate, and remove instances.
* Use separate game directories for complete isolation.
* Launch any instance with any available account.
* Configure Java, memory allocation, JVM arguments, and other launch settings individually.

### Powerful Mod Manager

Managing mods shouldn't be a hassle.

* Detect and identify installed mods automatically.
* Check for updates directly from Modrinth.
* Install missing dependencies with a single click.
* Remove duplicate files automatically.
* Detect incompatible mods based on your Minecraft version and mod loader.
* Export and import mod presets to quickly recreate your setup.

### Discover Mods & Resource Packs

Browse Modrinth without leaving the launcher.

* Search thousands of mods and resource packs.
* Filter by Minecraft version, mod loader, categories, and more.
* Install content directly into your selected instance.

### Account Management

Keep all your accounts in one place.

* Add and manage offline accounts.
* Instantly switch between accounts before launching.

### Personalize Everything

Make the launcher feel like your own.

* Customize accent colors and the overall theme.
* Use your own background images.
* Adjust blur, dimming, tint, and image scaling.
* Enable transparent interface effects.
* Use a custom in-app title bar.
* Choose custom fonts.

### Performance Controls

Optimize every instance for your hardware.

* Set custom RAM allocation.
* Configure JVM arguments.
* Select a custom Java installation.

### Privacy & Security

Your data stays yours.

* Hide usernames throughout the interface.
* Automatically redact sensitive information from logs.
* Remove Minecraft session tokens from console output.
* Clear account sessions when exiting the launcher.

### Built-in Console

Follow everything happening in real time with a detailed, colorized console, including launcher events and Minecraft output.

### Download Manager

Monitor every active download from a single place with progress tracking and status updates.

### Dawn Client Integration

Install or remove Dawn Client from any compatible instance in just a few clicks.

### System Tray Support

Keep the launcher running quietly in the background while your game is open.

### Designed for a Better Experience

Zero Launcher is built with smooth animations, thoughtful interactions, and a clean interface that stays out of your way so you can focus on playing.

# زيرو لانشر

لانشر ماينكرافت حديث مبني بالكامل باستخدام Java (Swing/AWT)، يركز على السرعة، والخفة، وسهولة الاستخدام، مع توفير تجربة متكاملة لإدارة الإصدارات والمودات دون تعقيد.

## المميزات

### إدارة الإصدارات

* إنشاء وتعديل ونسخ وحذف الإصدارات بسهولة.
* استخدام مجلد لعبة مستقل لكل إصدار.
* تشغيل أي إصدار بأي حساب متوفر.
* تخصيص إعدادات Java، والذاكرة، ووسائط JVM لكل إصدار بشكل مستقل.

### إدارة المودات

إدارة المودات أصبحت أبسط.

* التعرف على المودات المثبتة تلقائيًا.
* البحث عن التحديثات مباشرة من Modrinth.
* تثبيت التبعيات المفقودة بضغطة واحدة.
* إزالة الملفات المكررة تلقائيًا.
* اكتشاف المودات غير المتوافقة مع إصدار اللعبة أو محمل المودات.
* تصدير واستيراد مجموعات المودات لإعادة استخدامها أو مشاركتها.

### استكشاف المحتوى

تصفح محتوى Modrinth مباشرة من داخل اللانشر.

* البحث بين آلاف المودات وحزم الموارد.
* فلاتر حسب إصدار ماينكرافت، ومحمل المودات، والتصنيفات وغيرها.
* تثبيت المحتوى مباشرة داخل الإصدار الذي تختاره.

### إدارة الحسابات

* إضافة وإدارة حسابات الأوفلاين.
* التبديل بينها بسهولة قبل تشغيل اللعبة.

### تخصيص الواجهة

خصص اللانشر بالطريقة التي تناسبك.

* تغيير ألوان الواجهة والثيم.
* استخدام صور خلفية مخصصة.
* التحكم في التمويه، والتعتيم، والتلوين، وطريقة عرض الخلفية.
* دعم تأثيرات الشفافية.
* شريط عنوان مخصص داخل التطبيق.
* إمكانية استخدام خطوط مخصصة.

### إعدادات الأداء

* تخصيص حجم الذاكرة لكل إصدار.
* إضافة وسائط JVM مخصصة.
* اختيار نسخة Java التي تريد استخدامها.

### الخصوصية والأمان

* إخفاء اسم المستخدم داخل الواجهة.
* إزالة المعلومات الحساسة من السجلات تلقائيًا.
* إخفاء رموز الجلسات من سجل التشغيل.
* حذف جلسات الحسابات عند إغلاق اللانشر.

### سجل التشغيل

عرض سجل مفصل وملون لجميع العمليات، بما في ذلك مخرجات اللعبة، لتسهيل المتابعة وتشخيص المشاكل.

### إدارة التنزيلات

متابعة جميع التنزيلات الجارية من مكان واحد مع عرض الحالة ونسبة التقدم.

### تكامل Dawn Client

تثبيت أو إزالة Dawn Client لأي إصدار متوافق بسهولة.

### دعم System Tray

يمكن إبقاء اللانشر يعمل في الخلفية أثناء تشغيل اللعبة دون إزعاج.

### تجربة استخدام محسنة

واجهة نظيفة وحديثة مع انتقالات سلسة واهتمام بالتفاصيل، لتكون إدارة ماينكرافت والمودات تجربة بسيطة وسريعة.
